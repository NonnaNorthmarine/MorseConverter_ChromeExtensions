# MorseConverter_ChromeExtensions
## 概要
モールス信号を変換するChrome拡張機能です。
Geminiを仕様して作成しました。
https://gemini.google.com/share/3ba96b54332f

## 使用方法
1. ダウンロードしたzipを解凍し、任意のディレクトリに格納してください。
2. Chromeから `chrome://extensions/` を開く。
3. 右上の「デベロッパーモード」をONにする。
4. `パッケージ化されていない拡張機能を読み込む` から、1のディレクトリを選択。
5. ピン留めをすると使いやすいです。
